系统部署参考

  1. Apache 服务器配置

    网站前台域名根目录请指向 /public/ ，网站后台域名根目录请指向 /admin/。

  2. 数据库配置

    在/dbConf.php 文件里配置数据库用户名密码等信息

    通过 /docs/db/backup/backup.xxx.sql文件，将数据导入数据库。 注意，你可能需要修改该文件中的DB名称，当前是`test`库。

  3. 目录权限配置

    为了文件能够正常上传，请将如下目录权限设为Apache可写。

    • /public/image/adver/
    • /public/image/house/
    • /public/image/imgad/
    • /admin/uploads/image/adver/
    • /admin/uploads/image/house/
    • /admin/uploads/image/imgad/

  4. 网站配置文件

    部署时请务必要根据自己的部署环境调整/config.php 中的:

    • // 本地主机名称
    • $_cfg_hostName = 'gh';
    • // 服务器上的前台网站根目录
    • $_cfg_webRoot = "/var/www/proj/gh/public/";
    • // 服务器上的后台网站根目录
    • $_cfg_webRootAdmin = "/var/www/proj/gh/admin/";
    • // 网站前台的http根目录
    • $_cfg_siteRoot = "http://abc.abc.abc/";
    • // 网站后台的http根目录
    • $_cfg_siteRootAdmin = "http://bcd.bcd.bcd/";
